::-moz-color-swatch

representa el color seleccionado en un input de type="color".

::-moz-progress-bar

representa la barra de progreso dentro de un elemento. (La barra representa la cantidad de progreso que se ha hecho).

::-moz-range-progress

El pseudoelemento CSS es una extensión de Mozilla que representa la parte inferior de la pista (es decir, el surco) en la que el indicador se desliza en forma de . Esta porción corresponde a valores más bajos que el valor actualmente seleccionado por el pulgar (es decir, perilla virtual)

::-webkit-inner-spin-button

Se usa para diseñar la parte interna del botón giratorio de los elementos de entrada del selector de números.

::-webkit-meter-even-less-good-value

elemento cuando el valor y los atributos óptimos se encuentran fuera del rango bajo-alto, pero en zonas opuestas. Para ilustrar, se aplica cuando valor < bajo < alto < óptimo o valor > alto > bajo > óptimo
